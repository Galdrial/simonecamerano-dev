Progetto in evidenza

Tic Tac Vue

Versione interattiva del famoso gioco Tic Tac Toe (Tris) realizzata con Vue.js per offrire un'esperienza di gioco fluida e coinvolgente direttamente nel browser.

Altri progetti

Green Eats - Vegetarian recipe website

Sito di ricette vegetariane realizzato con React.js e API esterne per fornire agli utenti un'esperienza culinaria sana e gustosa.

Owly - Book Search Application

Un'applicazione web moderna e responsive per la ricerca e l'esplorazione di libri per categoria che utilizza l'API di Open Library.

Moove

Questo progetto mira a sviluppare un sistema in TypeScript che modella la struttura organizzativa di Moove, un servizio innovativo di micromobilità condivisa, focalizzandosi sulle interazioni tra utenti, mezzi di trasporto e le città servite.

Digital Counter

Un'applicazione web counter digitale avanzata e responsive, costruita con JavaScript moderno, Bootstrap 5 e stili CSS personalizzati.

Hotel Pomelia

Landing page per un hotel ipotetico situato a Ragusa, Sicilia. Progetto realizzato con HTML5 e CSS3.
